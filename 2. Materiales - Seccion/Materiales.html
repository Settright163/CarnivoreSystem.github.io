Materiales y su Función en el Proyecto

1. Sensores y Dispositivos Electrónicos

Sensor de humedad del suelo

Mide el nivel de humedad en la tierra para determinar cuándo es necesario regar la planta.

Sensor de temperatura y humedad

Registra la temperatura y humedad del ambiente para analizar cómo afectan el crecimiento de la planta.

Módulo relé de 5V

Actúa como un interruptor para encender o apagar dispositivos como la bomba de agua.

Mini bomba de agua

Se encarga de suministrar agua a la planta cuando los sensores indican que el sustrato está seco.

Microcontrolador ESP32

Procesa la información de los sensores y controla los dispositivos del sistema (bomba de agua, relé, etc.).

2. Conectores y Circuitos

Cables y accesorios

Permiten la conexión entre sensores, actuadores y el microcontrolador para la transmisión de datos y energía.

Fuente de alimentación de 12V

Proporciona la energía necesaria para el funcionamiento del sistema.

Placa PCB Virgen

Se utiliza para diseñar un circuito impreso que haga el sistema más compacto y organizado.

3. Materiales Biológicos

Planta carnívora

Es el objeto de estudio del proyecto, se monitorea su crecimiento y reacción a factores ambientales.

Sustrato especializado para plantas carnívoras

Proporciona las condiciones ideales de suelo para el desarrollo adecuado de la planta.

4. Estructura y Montaje (Soporte y Protección del Sistema)

Madera MDF para el terrario

Sirve como base estructural para sostener los componentes y proteger la planta.

Acetato para el terrario

Actúa como cubierta transparente para mantener un microclima controlado y proteger la planta de factores externos.

5. Usos de los Softwares

Arduino IDE

Se usará para programar el ESP32 y controlar los sensores.

HTML y CSS

Servirán para crear la plataforma web donde se mostrarán los datos.

ESP32

Recolectará información de los sensores y la enviará a la web en tiempo real.
